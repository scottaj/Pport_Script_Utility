run_script(mod = 0) click to toggle source

Runs the current script. the mod variable determines what type of run is started:

mod:

  • mod==-1 is a dry run where no data is output

  • mod==0 is a normal run starting from the beginning of the script

  • mod==1 finds the currently selected command and starts the run from there.

  • mod==2 Is a looped run.
